#{XY} QuickDevelop

{XY} 是一个常用方法与常用第三方库的集合.

### 
XYSpriteView 帧动画精灵类.
### 
XYSpriteManager 精灵管理类.
### 
XYTimer 定时器类.默认只有一个定时.
### 
NetWorkEngine 网络通讯类.
### 
DataLite 轻量形数据持久化, 基于NSUserDefaults


## External 使用的第三方库
* 开启ARC: -fobjc-arc
* 无 ARC: -fno-objc-arc
* 参考了 [BeeFramework](https://github.com/gavinkwoe/BeeFramework/blob/master/document)
